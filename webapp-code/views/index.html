What's this?

The "Cooper-Rover" is a unique game where a car with a webcam mounted streams video to this webapp! Here, you can control the car. It's a video-game brought to life!

Why isn't it working?

Currently, this application works for only one car-webcam-raspberry pi 2-spark photon set up. If you have been verified to have that particular setup, you are ready to play! Soon, this application will take one step-by-step to creating their very own car-webcam-raspberry pi 2-spark photon set up, so you can play as well!
